# Справочник объектов поиска и анализа
При работе с клиентами и их платежами требуется выполнять множество проверок данных на совпадение с определенными объектами.
Например, наличие указанных счетов, номеров карт, email, телефонов, ИНН и пр.
Справочник объектов поиска и анализа предназначен для хранения таких объектов. 
Основные возможности:
- ведение справочника типов объектов, заведение/редактирование/удаление типов объектов в ручном режиме;
- ведение справочника объектов, взаведение/редактирование/удаление объектов в ручном режиме;
- автоматическое заполнение справочника объектов путем анализа указанной строки и регулярного выражения для поиска;
- анализ произвольной строки на совпадение с объектом из справочника:
	acc407p_utl_graf.check_str(pStr IN VARCHAR2, pType IN VARCHAR2 DEFAULT 'ALL') 
	параметры:
	pStr – строка для анализа (назначение платежа, счет получателя);
	pType – тип данных (ACC,CARD,..), если не указано – то поиск будет по всем типам.
	При совпадении функция вернет само свопадение, иначе - пусто.
